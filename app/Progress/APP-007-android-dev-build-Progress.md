## What / why
A real, standalone Android APK — no Expo Go, no store, no EAS cloud. CNG:
`android/` is generated by `expo prebuild` and stays gitignored; all native config
lives in `app.config.ts` + config plugins. Unlocks the real expo-notifications
path, the Health Connect native module (APP-038), and `vita://` deep-link sign-in.

## Build recipe (one command each)
Prereqs: Android SDK at `~/Library/Android/sdk`, Java 21, `android/local.properties`
with `sdk.dir` (written by us; gitignored).

    cd app/services/vita-app
    npx expo prebuild --platform android --no-install --clean   # regenerates android/
    cd android && ANDROID_HOME=~/Library/Android/sdk ./gradlew :app:assembleRelease

- **mock-mode APK** (offline, seeded demo): build with `VITA_API_BASE_URL` unset.
- **prod-backed APK**: `VITA_API_BASE_URL=https://y9d7tlqsnl.execute-api.eu-west-1.amazonaws.com/v1 ./gradlew :app:assembleRelease`
  (the release JS-bundle task evaluates `app.config.ts`, baking `extra.apiBaseUrl`).
